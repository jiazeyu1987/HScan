### 🚨 严重问题（需要立即修复）

1. **API 404错误**
   - 招投标页面`fetchTenders`函数返回404
   - 仪表板`loadData`函数返回404错误
   - 影响：核心数据无法加载

2. **路由配置问题**
   - `/crawler`路径重定向到`/dashboard`
   - 缺少404错误页面

### ⚠️ 中等问题（建议优化）

3. **Ant Design组件废弃警告**
   - Card `bodyStyle` → 应使用`styles.body`
   - Input/InputNumber `addonAfter` → 应使用`Space.Compact`
   - Timeline.Item → 应使用`items`属性
   - Tabs.TabPane → 应使用`items`属性

### ℹ️ 轻微问题（可选修复）

4. **Message组件使用警告**
   - useForm未连接到Form元素
   - message静态函数上下文警告
